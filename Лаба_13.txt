1. Какое событие элемента управления Button обрабатывается в программах чаще всего?
Событие нажатия на элемент (Click).

2. Для чего предназначен компонент CheckBox? Назовите основные свойства класса CheckBox.
Компонент CheckBox прднахначен для установки флагов выбора элеменат в приложении.  Основные свойста:  
Checked, Unchecked, Indeterminate (неопределёное состояние, CheckBox становится серым).

3.  Опишите назачение элемента RadioButton. Какой внешний вид может принимать данный компонент? Назовите
основные свойства класса RadioButton.
RadiButton предназнаен для выбора действий в приложении; является переключателем. При выбранном в  
свойстве Appearance параметра Normal - RadioButton выглядит, как маленький кружочек с меткой рядом с
ним, при  выбранном параметре Button - выглядит, как стандартная кнопка.  Основные свойства класса
RadioButton: Checked, UnChecked.

4. Назовите классы компонентов для представления списочной информации.
ListBox, ComboBox.

5. Опишите основные свойтсва класса ListBox. Чем компонент ListBox отличается от checkedListBox?

ListBox - это компонент, который позволяет выбрать элемент в приложении	из предложенных вариантов,
а checkedListBox - это ListBox, в котором каждый выбранный элемент отмечается галочкой слева.
