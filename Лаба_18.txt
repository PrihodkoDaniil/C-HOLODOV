1. Опишите основные свойства класса Window.
Icon - иконка приложения, Title - заголовок окна, WindowStyle - стиль внешнего вида окна, WindowStartupLocation - место появления окна,
Height - высота окна, Width - ширина окна и др.

2. Что такое контейнер компоновки? Для чего они используются? Опишите свойства класса Panel.
Контейнер компановки - это специальзированный для размещения элементов управления компонент.
Свойства класса Panel: HorizontalAlignment(позиционирование по горизонтали), VerticalAlignment(позиционирование по вертикали), 
Margin(пространство вокруг объекта) и др.

3. Назовите основные контейнеры компоновки. Дайте характеристику и опишите особенности контейнеров компоновки.
StackPanel - размещает элементы в горизонтальном или вертикальном стеке; Grid - выстраивает элементы в строки и колонки невидимой таблицы;
Canvas - размещение элементов по фиксированным координатам.

4. Что такое присоединенное свойство? Приведите примеры.
Присоединённое свойство - это уникальное для элемента компоновки значение того или иного свойства. 
Например: <ColumnDefinition Width="10px"/>.

5. Какой контейнер применяется в окне приложения как контейнер компоновки по умолчанию?
Grid.

6. Какие свойства элементов управления влияют на положение элементов внутри контейнера компоновки?
VerticalAlignment, HorizontalAlignment, Margin.
