1. Назовите основные типы, необходимые дял работы с GDI+.
SolidBrush, Pen, Color, FontFamily, Graphics, Rectangle, Ellips, Triangle, Point и др.

2. Какие пространства имён доступны для работы с графикой?
System.Drawing, System.Drawing.Drawing2D и др.

3. Опишите возможные конструкторы класса Point.
Конструктор Point может принимать два значения - x и y, и одно значение - Int32 или Size. 

4. В чём состоит особенность конструктора класса Region?
Region представляет собой внутреннюю область фигуры, которой он инициализирован.

5. Как получить доступ к объекту класса Graphics? 
Создать экземпляр класса, проинициализировав его - имя_объекта.CreateGraphics().

6. Опишите способ, с помощью которого можно реализовать перехват события мыши над фигурами GDI+.
Для начала нужно создать экземпляры класса Region дял фигур, затем с помощью метода Is.Visible() передать координаты мыши по x и y. Если
координаты мыши будут в пределах региона, к которому мы применили данный метод, то будут выполнены те действия, которые мы укажем.

7. Для чего предназначен класс GraphicPath?
Для получения пути(доступа) к фигуре.

8. Чем отличаются типы Point и PontF?
Point принимает в качестве параметров целочисленный тип данных, а PointF - вещественный.

9. Какие элементы могут обрабатывать событие Paint?
Form, Panel.

10. Для чего используется тип PathPointType?
Для рисования сложных кривых линий.

11. Через какой тип происходит рисование в ообработчике события Paint?
Graphics.

12. Опишите механизм выбора объекта GDI.
Нужно создать флаг, который будет возвращать true, если курсор наведён на объект, что будет определяться в событии MouseMove методов
IsVisible().

13. Какие события участвуют в процессе интерактивного взаимодействия с GDI объектами графики?
Paint, Click, DoubleClick.

14. Какое событие необходимо задействовать, если добавить в программу отображение контура объекта при перетаскивании?
Paint.

15. Опишите механизм реализации анимации с использованием GDI+?
После изменение чего-то на форме (или другом объекте, на котором происходит анимация), её нужно перерисовать с помощью метода 
Invalidate().

16. Опишите назначение основных свойств и методов класса Timer.
Свойство Interval определяет временный промежуток одного тика таймера; методы Start() и Stop() запускают и останавливают таймер 
соответственно.

17. Для чего предназначено событие Tick класса Timer?
Для выполнение каких-либо действий при прохождении временного интервала одного срабатывания таймера.

18. Сколько таймеров может присутствовать в программе?
Сколько угодно.
